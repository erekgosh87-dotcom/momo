# Momo AI Agent - Project Identity

Momo is an **unlocked**, **cross-platform**, and **feature-enhanced** fork of the original Claude Code tool. It removes all telemetry, bypasses safety guardrails for experimental features, and provides a premium terminal experience.

## ✨ Core Features
- 🔓 **Unlocked Potential**: All internal experimental features (`ULTRATHINK`, `ULTRAPLAN`, `VOICE_MODE`, etc.) are enabled by default.
- 🚀 **Performance Optimized**: Built with [Bun](https://bun.sh) for lightning-fast startup and execution.
- 🎨 **Premium UI**: Custom high-tech splash screen and branding.
- 🛡️ **No Telemetry**: Completely private; no usage data is sent back to Anthropic.
- 💻 **Cross-Platform**: Ready for Windows, macOS, and Linux.
